# Bank-Term-Deposit-Prediction-Marketing-Campaign-Analysis
This project analyzes bank marketing campaign data to understand customer behavior and predict the likelihood of term deposit subscriptions. It uses Python for data cleaning, exploratory analysis, and machine learning model development. An interactive Power BI dashboard visualizes key performance metrics and customer segments.

- Built and evaluated machine learning models to predict term deposit subscription probability, improving targeting strategies.
- Designed an interactive Power BI dashboard to visualize KPIs such as conversion rates, campaign success, customer segments, and contact performance.
- Delivered data-driven insights to help optimize future marketing campaigns and enhance customer acquisition strategies.
